# IPL Auction PlayZone

A real-time multiplayer IPL auction simulation game where players can experience the thrill of building their dream cricket team.

> **Note**: This is a public showcase repository. The main project source code is maintained in a private repository.

![Status](https://img.shields.io/badge/Status-Active-success) ![Showcase](https://img.shields.io/badge/Type-Showcase-blue)

## About

IPL Auction PlayZone is an interactive web-based game that simulates the Indian Premier League player auction experience. Players can join rooms, bid on cricket players, and compete to build the best team within their budget.

### How to Play

1. **Sign Up/Login** - Create your account or sign in
2. **Create or Join Room** - Start a new auction or join an existing room with a code
3. **Wait for Players** - The room host can start when everyone's ready
4. **Live Bidding** - Bid on players as they come up for auction
5. **Build Your Team** - Manage your budget wisely and create the strongest squad
